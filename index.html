Materiales que se usarán para realizar esta página
01
CSS Básico

Es el código que usaremos para dar estilo a lo esencial de nuestra página, como por ejemplo, de que color quiero el texto, que estilo de letra quiero usar, cuánto margen le daré al texto...

02
CSS Avanzado

Es el código que usaremos para dar estilo al resto de la página, es decir, si quiero que la página se vea centrada, crear diferentes medidas para diferentes tipos de dispositivos...

03
Bootstrap

Es un framework CSS y Javascript diseñado para la creación de interfaces limpias y con un diseño responsive. Ofrece un amplio abanico de herramientas y funciones.

04
Tailwind

Es una potente herramienta para el desarrollo frontend. Permite a los desarrolladores y diseñadores ampliar estilos a los sitios web de una manera ágil y optimizada.
